# MoinLog WMS SaaS Landing Page - Installationsanleitung

## Überblick

Dieses Template ist ein vollständiges, professionelles SaaS-Landing-Page-Template für WordPress mit Elementor. Es wurde speziell für Warehouse Management System (WMS) Software entwickelt, kann aber leicht für andere SaaS-Produkte angepasst werden.

## Voraussetzungen

Bevor Sie beginnen, stellen Sie sicher, dass Sie Folgendes haben:

1. **WordPress Installation** (Version 5.0 oder höher)
2. **Elementor Pro Plugin** (Version 3.0 oder höher)
   - Das Template nutzt Pro-Features wie Custom CSS, Popup Builder und Theme Builder
3. **Admin-Zugang** zu Ihrer WordPress-Installation

## Installation

### Schritt 1: Elementor Pro installieren

1. Melden Sie sich in Ihrem WordPress-Admin-Dashboard an
2. Navigieren Sie zu **Plugins → Neu hinzufügen**
3. Suchen Sie nach "Elementor" und installieren Sie:
   - Elementor Website Builder (kostenlos)
   - Elementor Pro (Premium - erforderlich)
4. Aktivieren Sie beide Plugins

### Schritt 2: Template importieren

#### Methode A: Über Elementor Template Library (empfohlen)

1. Gehen Sie zu **Templates → Saved Templates** im WordPress-Dashboard
2. Klicken Sie auf **Import Templates**
3. Wählen Sie die Datei `templates/saas-landing-page.json` aus
4. Klicken Sie auf **Jetzt importieren**
5. Warten Sie, bis der Import abgeschlossen ist

#### Methode B: Über Elementor Page Builder

1. Erstellen Sie eine neue Seite in WordPress
2. Klicken Sie auf **Mit Elementor bearbeiten**
3. Klicken Sie auf das Ordner-Symbol (Template-Bibliothek)
4. Wählen Sie **Importieren** → **Datei auswählen**
5. Laden Sie `templates/saas-landing-page.json` hoch
6. Klicken Sie auf **Importieren**

### Schritt 3: Template anwenden

1. Öffnen Sie die Seite, auf der Sie das Template verwenden möchten
2. Klicken Sie auf **Mit Elementor bearbeiten**
3. Klicken Sie auf das Ordner-Symbol
4. Suchen Sie nach "SaaS Landing Page - MoinLog WMS"
5. Klicken Sie auf **Einfügen**

### Schritt 4: Template anpassen

Nach dem Import können Sie das Template nach Ihren Wünschen anpassen:

1. **Texte ändern**: Klicken Sie auf beliebige Textelemente und bearbeiten Sie den Inhalt
2. **Farben anpassen**: Nutzen Sie die Elementor-Farbauswahl für Ihre Brand-Farben
3. **Bilder hinzufügen**: Ersetzen Sie Platzhalter durch Ihre eigenen Bilder
4. **Buttons verlinken**: Fügen Sie Links zu Ihren CTAs hinzu (Demo-Anfragen, Registrierung, etc.)

## Anpassungsoptionen

### Farben ändern

Das Template verwendet folgendes Farbschema:

- **Primärfarbe (Dunkelblau)**: #1a1a2e
- **Sekundärfarbe (Mittelblau)**: #0f3460
- **Akzentfarbe (Türkis)**: #16c79a
- **Akzent Hover (Dunkeltürkis)**: #11a87e
- **Fehlerfarbe (Rot)**: #e74c3c
- **Hintergrund Hell**: #f8f9fa
- **Text Dunkel**: #1a1a2e
- **Text Mittel**: #555555
- **Text Hell**: #e0e0e0

Um Farben global zu ändern:
1. Gehen Sie zu **Elementor → Global Settings → Colors**
2. Definieren Sie Ihre Brand-Farben
3. Wenden Sie diese auf das Template an

### Schriftarten anpassen

1. Navigieren Sie zu **Elementor → Custom Fonts**
2. Laden Sie Ihre Brand-Schriftarten hoch
3. Weisen Sie diese in den **Global Settings → Typography** zu

### Responsive Design

Das Template ist vollständig responsive. Um mobile Ansichten anzupassen:

1. Klicken Sie auf das Responsive-Icon in Elementor
2. Wählen Sie Tablet oder Mobile
3. Passen Sie Abstände, Schriftgrößen und Layout an

## Best Practices

### Performance-Optimierung

1. **Bilder optimieren**: Komprimieren Sie alle Bilder vor dem Upload
2. **Lazy Loading**: Aktivieren Sie Lazy Loading für Bilder
3. **Caching**: Nutzen Sie ein Caching-Plugin wie WP Rocket
4. **CDN**: Verwenden Sie ein CDN für schnellere Ladezeiten

### SEO-Optimierung

1. **Meta-Titel**: Setzen Sie einen aussagekräftigen Seitentitel
2. **Meta-Beschreibung**: Fügen Sie eine optimierte Beschreibung hinzu
3. **Überschriften-Hierarchie**: Behalten Sie die H1-H4 Struktur bei
4. **Alt-Tags**: Fügen Sie allen Bildern Alt-Texte hinzu

### Conversion-Optimierung

1. **CTAs testen**: A/B-testen Sie verschiedene Button-Texte
2. **Social Proof**: Fügen Sie echte Kundennamen und Logos hinzu
3. **Vertrauen aufbauen**: Zeigen Sie Zertifikate und Auszeichnungen
4. **Geschwindigkeit**: Optimieren Sie Ladezeiten für bessere Conversion

## Fehlerbehebung

### Template wird nicht korrekt angezeigt

1. Stellen Sie sicher, dass Elementor Pro aktiviert ist
2. Überprüfen Sie, ob alle erforderlichen Widgets verfügbar sind
3. Leeren Sie den Cache (Browser und WordPress)
4. Aktualisieren Sie Elementor auf die neueste Version

### Import schlägt fehl

1. Erhöhen Sie die PHP-Upload-Grenze in der php.ini
2. Überprüfen Sie die WordPress-Dateiberechtigungen
3. Deaktivieren Sie temporär Sicherheits-Plugins
4. Nutzen Sie die alternative Import-Methode

### Icons werden nicht angezeigt

1. Gehen Sie zu **Elementor → Tools → Regenerate CSS**
2. Klicken Sie auf **Regenerate Files**
3. Leeren Sie den Browser-Cache
